# Kd-tree e o problema do vizinho mais proximo
## Sobre o Projeto
Este repositório armazena códigos, dados e materiais de estudo do projeto de pesquisa sobre Kd-tree e o problema do vizinho mais proximo.

Árvores k-d são estruturas de dados que realizam um particionamento do espaco com K-dimensões, permitindo realizar consultas rapidamente nos objetos examinados. Desse modo, ela se mostra como uma ferramenta util para resolver rapidamente o problema do vizinho mais proximo, na qual, dado um conjunto de dados, deseja-se encontrar o objeto mais similar ao da busca, seguindo algum meio de medicao pré definido.

## Roteiro
O desenvolvimento do projeto foi dividido em duas etapas, a primeira o estudo de conhecimentos fundamentais de algoritimos, estruturas de dados e geometria computacional. Na segunda parte, o aprofundamento em árvores kd, implementacao, e experimentos com bases de dados sinteticas.

###Etapa 1:
- Conceitos básicos em algoritmos, estruturas de dados e análise de algoritmos
- Árvores binárias de busca
- Geometria espacial
- Compreensao do problema do vizinho mais próximo

###Etapa 2:
- Introducão às árvores k-d
- Realizacão de experimentos em bases de dados: variando tamanho, distribuicão e dimensionalidade.
- Ánalise dos resultados obtidos
